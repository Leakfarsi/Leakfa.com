Leakfa is a free and secure service for checking personal data breaches of Iranian users — similar to the trusted [Have I Been Pwned](https://haveibeenpwned.com) service, but with special focus on security incidents related to Iran 🇮🇷.

---

## 📌 About the Project

### 💡 Why was Leakfa created?

Leakfa was created in response to the widespread wave of data breaches in April and May 2020, when:
- Many major Iranian organizations were targeted by cyber attacks
- Millions of personal data records were leaked
- Most organizations did not inform their users about these breaches
- Regular users had no way to check the security status of their information

### 🎯 Project Goals

- 🛡️ **Increase Security Awareness**: Help users understand the security status of their information
- 🔍 **Information Transparency**: Provide easy access to data breach information
- 📚 **Promote Cyber Literacy**: Education and promotion of best security practices
- ⚖️ **Accountability**: Encourage organizations to better protect user data

---

## 🔐 How It Works

### 🔄 Secure Search Process

1. **User Input**: Enter your email or mobile number
2. **Local Hashing**: Data is hashed with SHA-1 algorithm in your browser
3. **Secure Transfer**: Only the generated hash (not the original data) is sent to the server
4. **Search**: Server compares the hash with the breach database
5. **Result**: Your information security status is displayed

### 📂 Data Sources

- 📊 Confirmed public breaches
- 📧 Anonymous reports from trusted sources
- 🔍 Continuous monitoring of Iran's cyberspace
- ✅ Verification and validation of all data before addition

---

## 🔒 Privacy and Security

### 🛡️ Security Guarantees

- 🚫 **No raw data stored**: Only SHA-1 hashes are processed
- 🔐 **Advanced encryption**: All communications secured with HTTPS
- 🕵️ **Completely anonymous**: Searcher identity remains protected
- 📝 **Minimal logging**: Only general statistics for service improvement

### 💾 Stored Information

- ✅ SHA-1 hash of searched data
- ✅ IP address (for security and abuse prevention)
- ✅ Request timestamp
- ❌ No raw personal data
